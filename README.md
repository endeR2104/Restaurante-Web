## 📖 Descripción

Bienvenido al repositorio oficial del sitio web del restaurante **Sabor Elegante**. Este proyecto es una página web estática diseñada para reflejar la identidad de un restaurante de alta cocina, presentando su carta, historia y filosofía de una manera visualmente atractiva y elegante.

El sitio está desarrollado para ser una carta de presentación digital que inspire a los visitantes a reservar y disfrutar de una experiencia culinaria excepcional.

## ✨ Características Principales

*   **Diseño Responsive:** Adaptado para una visualización óptima en dispositivos móviles, tabletas y ordenadores de escritorio.
*   **Presentación Visual:** Muestra las especialidades del restaurante con imágenes atractivas (implementación sugerida) y descripciones detalladas.
*   **Secciones Claras:**
    *   **Hero:** Un llamado a la acción principal para reservar.
    *   **Especialidades:** Destaca los platos estrella.
    *   **Historia:** Narra la trayectoria del restaurante y su filosofía culinaria.
*   **Llamada a la Acción:** Botones estratégicos para fomentar la interacción del usuario (ej. "Reserva ahora", "Conoce más").
*   **Código Limpio:** Estructura HTML y CSS bien organizada, fácil de mantener y escalar.
